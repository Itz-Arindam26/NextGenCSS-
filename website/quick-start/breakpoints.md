# Breakpoints

## Responsive size

_Responsive design size are categorized into 3 different categories_

**`Mobile`, `Tablet`, `Desktop`**

* **`Mobile` : between `0px` and `767px`**
* **`Tablet` : between `768px` and `991px`**
* **`Desktop` : between  `992px` and ...**
